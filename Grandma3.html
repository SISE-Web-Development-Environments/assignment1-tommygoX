About Me


My name is Agatha Feuerweker and I am 87 years old.
I was born in Romania in a town called Timishwara.
I moved my family to Israel in 1956 but the situation was too difficult so we moved again ,
this time to Brazil.
40 years later I decided to come back to our holly land and so me and my husband got back to Israel ,
this time for good.

Q&A



1.Web 1.0 refers to the first stage of the World Wide Web evolution.
In Web 1.0 advertisements on websites while surfing the internet is banned.
Also, in Web 1.0, Ofoto is an online digital photography website,
on which user could store, share, view and print digital pictures.
Web 2.0 refers to world wide website which highlight user-generated content, usability and interoperability for end users.
Web 2.0 includes Dynamic content that is responsive to user input,
Developed APIs to allow self-usage, such as by a software application and
Free sorting of information, permits users to retrieve and classify the information collectively.
Web 3.0 - It refers the evolution of web utilization and interaction which includes altering the Web into a database.
Web 3.0 main features:
Semantic Web , Artificial Intelligence and Ubiquity.
